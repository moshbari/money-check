# 💰 The Money Check

A plain-English tool that answers one question about any business idea:
**"Will this actually make money?"**

Answer 3 quick questions — what a customer pays you, how many times they'll buy,
and what it costs to get them — and get an instant, jargon-free verdict:
🔴 Losing · 🟠 Risky · 🟡 Healthy · 🟢 Scaling.

It also shows what one customer is really worth to you, how fast you earn back
what you spend, the 6 easy ways to make more, and the "pick a lane" rule.

## Live site

Once GitHub Pages is turned on, the app will be live here:

> `https://<your-username>.github.io/money-check/`

## Run it locally

Just open `index.html` in any web browser. No build step, no dependencies —
it's a single self-contained file.
